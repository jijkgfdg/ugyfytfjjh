# 日本麻將拍照算點 PWA

這是可直接安裝到安卓主畫面的版本。

## 立刻使用
1. 解壓這個資料夾
2. 把裡面的所有檔案上傳到任何 **HTTPS 靜態網站**
3. 用安卓 Chrome 開啟 `index.html`
4. 按頁面上的 **加入主畫面**
5. 第一次開相機時按 **允許**

## 檔案說明
- `index.html`：主程式
- `manifest.json`：PWA 安裝資訊
- `sw.js`：離線快取
- `icons/`：主畫面圖示

## 注意
- 本地直接打開 `index.html` 時，人工計分可用，但相機通常受瀏覽器安全限制。
- 真正的牌面自動辨識模型尚未接入；這版主打你可以現在就拍照、上傳、手動修正、算點、安裝到桌面。
